# 4択クイズ (スマホ向け・ローカル動作)

- このフォルダ: `Desktop/work/90_cc/20250923/codex-quiz4`
- エントリ: `index.html`
- スマホ前提のUIで、1ラウンド=5問。履歴と累積スコアをローカル保存。

## 使い方

- `index.html` をブラウザで開く
  - そのまま内蔵サンプル問題で遊べます
  - JSON/YAML ファイルで外部の問題セットを読み込み可能
- メニューで「クイズ開始」で5問出題
- 結果は「スコア履歴」に保存され、累積スコアと挑戦回数がメニューに表示

## 問題セットの形式

どちらもOK: JSON または YAML。
- JSON 例: `data/questions.json`
- YAML 例: `data/questions.yml`

構造は以下いずれか。

- 直接配列
[
  {
    "category": "化学",
    "question": "水の化学式はどれ？",
    "choices": ["HO", "H2O", "H2O2", "OH2O"],
    "answer": 1,
    "explanation": "水はH2O"
  }
]

- オブジェクトの `questions` 配列
{
  "questions": [ { ...同上... } ]
}

フィールド名
- `category` 文字列 (任意)
- `question` または `text` 文字列 (必須)
- `choices` または `options` 文字列配列 (必須・4択想定)
- `answer` または `correctIndex` または `correct` 数値 (0始まり)
- `explanation` 文字列 (任意)

YAML はインデント2スペース想定のシンプル構文を想定しています。複雑なアンカー/複合スカラ等は非対応です。

## スコア保存

- ブラウザの `localStorage` に保存
- 累積: 正解数/総出題数、挑戦回数
- 履歴: 各ラウンドの日時・スコア・セット名（最大200件）

## 注意

- ファイルを直接開いて動作します（ローカルサーバ不要）。
- 外部ファイルの読み込みは「問題セットを読み込む」から行ってください（`fetch`は使っていません）。
